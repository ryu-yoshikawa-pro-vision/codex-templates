# Plan

## Objective
- ユーザーが「計画を立てて」と依頼した際、`docs/plans/{yyyy-mm-dd}_{プラン名}.md` 形式で計画書を作成する運用を AGENTS.md とテンプレートに明記する。

## Scope
- In:
  - ルート `AGENTS.md` への運用ルール追加
  - `.codex/templates/` 配下テンプレートの更新
  - 必要に応じた `docs/PROJECT_CONTEXT.md` 追記
- Out:
  - 実行エンジン実装や自動生成スクリプトの追加

## Assumptions
- 対象はこのリポジトリの運用ドキュメント整備であり、アプリコード変更は不要。

## Approach
- 現行ルールとテンプレートを確認し、計画依頼時の保存先・命名規則を一貫して追記する。
- 変更後に差分確認と最低限の整合チェックを行う。

## Definition of Done
- `AGENTS.md` に計画書作成ルールが明記されている。
- テンプレートに同ルールが反映され、次回以降の run 作成時にも利用できる。
- 変更内容がコミットされ、PR メッセージが作成されている。

## Risks / Unknowns
- 既存ルールとの重複記述により冗長化する可能性。
