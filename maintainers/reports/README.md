# maintainers/reports Usage

## Purpose
- Store source-repo investigation notes, execution logs, and analysis reports.

## Naming Rule
- Create report files as: `{yyyy-mm-dd}_{HHMMSS}_{report_name}.md`
- Use JST (`Asia/Tokyo`) for the date/time portion.

## Example
- `maintainers/reports/2026-03-14_180000_v2-validation.md`
